Visualize Your SQL Dependencies

An interactive graph visualization tool for Microsoft SQL Server, Azure Synapse, and Fabric. Automatically extract and analyze lineage from stored procedures without writing a single line of Python.

Try Live Demo Run Locally (Docker) View on GitHub

One-click local deployment:

docker run -d -p 8000:8000 -v data-lineage-config:/app/config --name data-lineage chwagneraltyca/data-lineage-visualizer:latest

Key Features

Built with React Flow for interactive graph visualization

YAML-Based Parser

Business-maintainable extraction rules without Python knowledge

Microsoft Stack Ready

SQL Server, Azure SQL, Synapse Analytics, and Fabric

5-Minute Setup

Database direct connection or offline Parquet upload
